### 2. Новые модели в Prisma (12 моделей)

| Модель | Описание |
|--------|----------|
| `GlobalSettings` | Глобальные настройки (язык, AI, производительность, безопасность) |
| `PlatformSettings` | Настройки платформ (Telegram, Instagram, TikTok, YouTube, Twitter) |
| `CampaignSettings` | Настройки кампаний (контент, автоматизация, A/B тесты) |
| `PostSettings` | Настройки постов (переопределения) |
| `NotificationSettings` | Настройки уведомлений |
| `SettingsPreset` | Шаблоны/пресеты настроек |
| `BlacklistSettings` | Чёрные списки |
| `AutomationRule` | IF-THEN правила автоматизации |
| `PublishingSettings` | Настройки публикации |
| `AnalyticsSettings` | Настройки аналитики |
| `DevSettings` | Настройки разработчика |
| `HotkeysSettings` | Горячие клавиши |

## 🚀 Как использовать

### Открыть настройки
Клик по иконке шестерёнки в интерфейсе

### Структура диалога
```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│  ⚙️ НАСТРОЙКИ                                            [Поиск: ___________] [⊗] │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│  ┌─────────────────────────────────────────────────────────────────────────────────┐│
│  │ [🌍 Система] [📱 Платформы] [🎨 Контент] [🤖 AI-агент] [⚡ Фичи] [📊 Аналитика] ││
│  └─────────────────────────────────────────────────────────────────────────────────┘│
│                                                                                     │
│  ... контент вкладки ...                                                            │
│                                                                                     │
│  ───────────────────────────────────────────────────────────────────────────────── │
│  [Сбросить всё]  [Экспорт]  [Сохранить как пресет]  [Применить]  [Отмена]           │
└─────────────────────────────────────────────────────────────────────────────────────┘
```

### Примеры IF-THEN правил
```
ЕСЛИ пост набрал >1000 просмотров за час ТО опубликовать на всех платформах
ЕСЛИ пост набрал <10 просмотров за час ТО удалить его
ЕСЛИ закончились бесплатные токены DeepSeek ТО переключиться на резервный AI
ЕСЛИ сегодня понедельник ТО создать пост "Итоги недели"
```
