================================================================================
ЗАПУСК СИСТЕМЫ
================================================================================

1. Веб-сервер (Next.js):
---------------------------------------------------------------------------
cd /home/z/my-project
bun run dev                    # Development mode
bun run build && bun run start # Production mode

2. Desktop Runner (Python):
---------------------------------------------------------------------------
cd /home/z/my-project/desktop-runner
pip install -r requirements.txt
export TELEGRAM_API_ID=your_id
export TELEGRAM_API_HASH=your_hash
python runner.py

3. Требования:
---------------------------------------------------------------------------
- Node.js 18+ или Bun
- Python 3.10+
- SQLite
- ADB (Android Debug Bridge) - для работы с устройствами
