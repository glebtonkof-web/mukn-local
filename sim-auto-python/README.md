# SIM Auto-Registration PRO

Полностью автоматическая система регистрации, прогрева и монетизации аккаунтов на 22+ платформах.

## Возможности

### 🎯 Регистрация
- **22+ платформы**: YouTube, TikTok, Instagram, Twitter, Telegram, Discord, Twitch, LinkedIn, Reddit, Facebook, VK, WhatsApp, Odnoklassniki, Spotify, Tinder, Snapchat, Pinterest, OnlyFans, Likee, Trovo, Rumble, Odysee
- **SMS верификация** через ADB (Android Debug Bridge)
- **Антидетект**: уникальные fingerprint, случайные задержки, человекоподобное поведение

### 🔥 Прогрев
- **4 фазы прогрева**: Ghost → Observer → Active → Profit
- **Платформо-специфичные стратегии** для Instagram (21 день), TikTok (28 дней), Twitter (14 дней)
- **Автоматическое управление рисками**: лимиты действий, паузы, расписание

### 💰 Монетизация
- **CPA схемы**: казино, дейтинг, нутра
- **Партнёрки**: криптобиржи, курсы
- **Фарминг**: создание и продажа аккаунтов
- **Арбитраж трафика**
- **Рекламные посты**

## Установка

### 1. Системные требования

```bash
# Ubuntu/Debian
sudo apt install adb python3 python3-pip

# Установка Playwright браузеров
playwright install chromium
```

### 2. Python зависимости

```bash
cd sim-auto-python
pip install -r requirements.txt
```

### 3. Настройка телефона

1. Включите **Отладку USB** в настройках разработчика
2. Подключите телефон через USB кабель
3. Разрешите отладку USB на телефоне

### 4. Проверка ADB

```bash
adb devices
# Должен показать список подключённых устройств
```

## Использование

### Список платформ

```bash
python register_all.py --list
```

### Сканирование SIM-карт

```bash
python register_all.py --scan
```

### Регистрация на всех платформах

```bash
python register_all.py
```

### Регистрация на конкретных платформах

```bash
python register_all.py -p telegram instagram tiktok
```

### Скрытый режим (без GUI браузера)

```bash
python register_all.py --headless
```

### С прокси (для заблокированных в РФ)

```bash
python register_all.py --proxy socks5://user:pass@host:port
```

### Схемы монетизации

```bash
python register_all.py --schemes
```

## Анти-бан техники

1. **Человекоподобные задержки**: нормальное распределение, не равные интервалы
2. **Кривые Безье для мыши**: естественное движение курсора
3. **Случайная скорость набора**: 0.02-0.12 сек между символами
4. **Паузы между действиями**: имитация "обдумывания"
5. **Случайные сессии**: разное время дня, разная длительность
6. **Уникальные fingerprint**: Canvas, WebGL, User-Agent

## Прогрев аккаунтов

### Фазы прогрева (на примере Instagram)

| Фаза | Дни | Действия | Сессии/день |
|------|-----|----------|-------------|
| Ghost | 1-5 | Просмотр, скролл | 2-4 |
| Observer | 6-10 | + лайки, подписки | 3-5 |
| Active | 11-21 | + комментарии | 3-6 |
| Profit | 22+ | + публикации | 4-8 |

### Риск-лимиты (Instagram)

- Максимум лайков/час: 30
- Максимум подписок/час: 15
- Максимум комментариев/час: 5
- Минимум задержка между действиями: 30 сек

## Рекомендации

1. **Не более 3 аккаунтов** на одну SIM-карту
2. **Прогрев минимум 14 дней** перед монетизацией
3. **Уникальный proxy** для каждого аккаунта
4. **Не более 150 действий в день** для новых аккаунтов
5. **Пауза 3-6 минут** между регистрациями разных сервисов
